## Local project rule

For this project only (`rc-web-clients`):

- Do **not** push to GitHub automatically after changes.
- Commit locally when useful, but **push only when the user explicitly asks**.
- Treat any push as requiring a direct user command such as:
  - "push"
  - "запушь"
  - "отправь в github"

This rule is local to this repository and does not apply globally to other projects.
